# Ice Island Forecast Dashboard

Weather forecast, station water temperature monitoring and CRREL's Heat Flux Tool in one dashboard for Sault Sainte Marie, MI.

Access at: https://iceislandforecast.streamlit.app/

## Features
- 7-day weather forecast from National Weather Service
- Real-time water temperature from NOAA stations
	- Interactive temperature trend charts
	- Station status monitoring
- CRREL Heat Flux Forecast tool (https://github.com/ski907/hff)
	- Adjustable temperatrue and characteristic depth
